## Key Concepts & Terms

### Security Concepts
- **Confidentiality**: Keeping information private and protected
- **Integrity**: Ensuring information is accurate and hasn't been modified
- **Availability**: Ensuring systems and information are accessible
- **Authentication**: Verifying identity
- **Authorization**: Determining access permissions
- **Non-repudiation**: Proving who did what

### Networking Terms
- **IP Address**: Internet Protocol address for device identification
- **Port**: Virtual endpoint for network communication
- **Protocol**: Set of rules for network communication
- **Firewall**: Network security device
- **VPN**: Virtual Private Network for secure communication
- **DNS**: Domain Name System for name resolution
- **HTTP/HTTPS**: Web communication protocols

### System Concepts
- **Kernel**: Core of operating system
- **Process**: Running instance of a program
- **User Privilege**: Access level permissions
- **File System**: Method of organizing data storage
- **Virtual Machine**: Simulated computer system
- **Cloud Service**: Internet-based computing resources

### Web Infrastructure
- **Client-Server**: Architecture where clients request from servers
- **Web Server**: Computer that hosts websites
- **Database**: Organized data storage
- **Load Balancer**: Distributes network traffic
- **CDN**: Content Delivery Network for faster content access
- **API**: Application Programming Interface
