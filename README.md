# Pre Security Learning Path Notes

Welcome to my comprehensive notes repository for the **TryHackMe Pre Security Path**! 🔐

This repository contains detailed, structured notes covering all modules from the Pre Security Path on TryHackMe. Whether you're starting your cybersecurity journey or looking for a handy reference guide, these notes have you covered.

## 📚 What's Inside

This repo is organized into **6 progressive modules** that build foundational knowledge in cybersecurity:

1. **Offensive Security Introduction** - Understanding the offensive side of security
2. **Computer Fundamentals** - Hardware, systems, and computing basics
3. **Operating Systems** - Windows and Linux essentials
4. **Networking Fundamentals** - Protocols, models, and connectivity
5. **Web Infrastructure** - DNS, HTTP, and how the web works
6. **Security Foundations** - CIA Triad and core security principles

Each module contains detailed markdown notes with explanations, concepts, and key takeaways.
